## How to deploy this app to the Expo?

### Step 1: Create an account
  [Sign up Expo](https://expo.dev/signup)

### Step 2: Deploy our application
```
sudo npm install -g expo-cli
```

### Step 3: Logging in
```
expo login
```
  >   Enter username or email

  >   Enter password
```
expo publish!
```
